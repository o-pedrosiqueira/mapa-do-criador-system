Política de Privacidade

Esta política descreve como o Mapa do Criador trata os dados pessoais de quem visita o site, compra o produto e usa o sistema local.

Última atualização: 19 de maio de 2026

1. Quem somos

O Mapa do Criador é um produto digital criado e mantido por Pedro Siqueira, pessoa física, criador de conteúdo e desenvolvedor do sistema.

Para questões sobre privacidade, entre em contato pelo e-mail contato@mapadocriador.com (substitua pelo e-mail real antes do lançamento).

2. Dados que coletamos

2.1. No site (página de vendas)

  • Cookies de medição (Meta Pixel, Google Analytics). Identificam visitas, origem do tráfego e comportamento agregado. Não armazenam dados pessoais identificáveis.
  • Endereço IP e User-Agent, registrados temporariamente pelo provedor de hospedagem (Vercel) para fins de segurança e operação.

2.2. No checkout

O processamento de pagamento é feito pela Hotmart. Os dados pessoais (nome, e-mail, CPF, endereço de cobrança, dados de pagamento) são coletados e armazenados pela Hotmart, regidos pela política de privacidade dela: hotmart.com/pt-br/privacy.

Recebemos da Hotmart apenas: nome, e-mail e confirmação de compra. Não temos acesso aos dados de pagamento.

2.3. No sistema instalado localmente

Todos os dados criados dentro do Mapa do Criador (DNA Criativo, Anthem, Arquétipo, Pilares, Manifesto, Caixa de Entrada, newsletters, carrosséis, stories) ficam exclusivamente no computador do aluno, em ~/Documents/mapa-do-criador/meus-produtos/. Não enviamos esses dados para nenhum servidor externo.

O sistema usa a API do Claude (Anthropic) para gerar conteúdo. Os prompts enviados à API são processados pela Anthropic conforme a política de privacidade deles: anthropic.com/legal/privacy.

3. Como usamos seus dados

  • Enviar o link de download e instruções de instalação após a compra (via e-mail automático da Hotmart).
  • Comunicar atualizações do sistema, novas versões e melhorias (eventualmente).
  • Responder dúvidas e oferecer suporte técnico.
  • Análise agregada do tráfego do site (volume de visitas, origem) para otimizar a divulgação.

4. Compartilhamento

Não vendemos, alugamos nem cedemos dados pessoais a terceiros. Dados são compartilhados apenas com:

  • Hotmart (processamento de pagamento, obrigatório para concretizar a compra)
  • Meta / Google (análise agregada via Pixel e Analytics, sem dados pessoais identificáveis)
  • Anthropic (provedor da API Claude usada pelo sistema instalado, apenas prompts gerados pelo aluno em uso ativo)

5. Seus direitos

Conforme a LGPD (Lei 13.709/2018), você tem direito a:

  • Confirmar se tratamos seus dados
  • Acessar seus dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar exclusão dos dados (exceto os exigidos por lei fiscal ou contratual)
  • Solicitar portabilidade
  • Revogar consentimento

Para exercer qualquer direito, escreva para contato@mapadocriador.com. Respondemos em até 15 dias úteis.

6. Segurança

O site é hospedado em infraestrutura criptografada (Vercel + Cloudflare). O checkout passa por HTTPS obrigatório com criptografia ponta a ponta. Dados do sistema local ficam no computador do aluno e dependem da segurança do dispositivo dele.

7. Cookies

Usamos cookies essenciais (sessão, segurança) e analíticos (Meta Pixel, Google Analytics). Ao continuar navegando, você consente com o uso de cookies. Para desabilitar, ajuste as configurações do seu navegador.

8. Alterações desta política

Esta política pode ser atualizada. A data de atualização aparece no topo. Mudanças significativas serão comunicadas por e-mail aos alunos cadastrados.
